Getting landcover data

In this example, we will look at landcover data for the proportion of urban/built area around Montréal; this dataset is very large to fit in memory, so we will take advantage of the ability to only load the part that matters by passing the limits of a bounding box.

using SimpleSDMLayers
urban = landcover(9; left=-74.0, right=-72.0, top=47.0, bottom=44.0)
SimpleSDMPredictor{UInt8}(Union{Nothing, UInt8}[nothing nothing … nothing nothing; nothing nothing … nothing nothing; … ; nothing nothing … nothing nothing; nothing nothing … nothing nothing], -74.00833333333333, -72.0, 44.00833333333334, 47.0)

This dataset is returning data as UInt8 (as it represents a proportion of the pixel), but this is not something that can be plotted efficiently. So in the next step, we will manipulate this object a little bit to have something more workable.

We can then fill these values using a simple rule:

for (i,e) in enumerate(urban.grid)
  n_urban_grid[i] = isnothing(e) ? NaN : float32(e)
end

And now we can rewrite our urban object as

urban = SimpleSDMPredictor(n_urban_grid, urban)
SimpleSDMPredictor{Float32}(Union{Nothing, Float32}[NaN32 NaN32 … NaN32 NaN32; NaN32 NaN32 … NaN32 NaN32; … ; NaN32 NaN32 … NaN32 NaN32; NaN32 NaN32 … NaN32 NaN32], -74.00833333333333, -72.0, 44.00833333333334, 47.0)

Note that this uses a shortcut where the bounding box from a new SimpleSDMLayer is drawn from the bounding box for a previous one.

heatmap(urban)
